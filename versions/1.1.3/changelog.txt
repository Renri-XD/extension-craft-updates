Extension Craft 1.1.3

The Login Fix update — first join no longer freezes the server!

- Fixed a 5-17 second server freeze (and possible timeout disconnect) when the first player joined after a server restart. Root cause: Sawmill's recipe sorting rebuilt every creative tab during login (a feature that does not work on NeoForge servers anyway) — now disabled via config (sort_recipes = false).
- Removed Log Begone — its per-line regex log filtering amplified the login freeze and hid the evidence.
- Removed Snow Real Magic — outdated blocks saved in chunks caused "Unknown registry key" errors and silent world data loss.
- Client/server sync scripts moved out of the pack (local tooling, not pack content).

Extension Craft 1.1.3

ログイン修正アップデート — 初回ログインでサーバーが固まらなくなりました！

- サーバー再起動後、最初のプレイヤーがログインした際に5〜17秒フリーズする（最悪タイムアウト切断される）問題を修正。原因は Sawmill のレシピ並び替えがログイン中に全クリエイティブタブを再構築していたこと（そもそもNeoForgeサーバーでは機能しない仕様）。config で sort_recipes = false に変更して無効化。
- Log Begone を削除 — 1行ごとの正規表現ログフィルタがフリーズを増幅し、証拠のログまで隠していたため。
- Snow Real Magic を削除 — チャンク内に残った旧バージョンのブロックが "Unknown registry key" エラーとワールドデータの静かな消失を引き起こしていたため。
- クライアント/サーバー同期スクリプトをパック外へ移動（ローカル運用ツールのため）。

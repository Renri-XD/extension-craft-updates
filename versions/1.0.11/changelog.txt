# Extension Craft 1.0.11

- FIX: The skill tree announced in 1.0.10 was never actually shipped — the mods were there, but the tree itself was missing, so the skills screen came up empty. The full tree is now included: five branches (Warrior, Vitality, Miner, Explorer, Farmer), 51 nodes, and a neutral starting node so your first point is yours to spend.
- FIX: The update checker kept saying "an update is available" even after updating to 1.0.10. Its version number was left behind at 1.0.9 — it now matches the pack again.
- NEW: The skill tree is fully translated in both English and Japanese.
- Sorry for the trouble on 1.0.10 — this release makes the skill tree actually work.
